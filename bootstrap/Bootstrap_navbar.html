2.向 nav 标签添加 class .navbar、.navbar-default。

3.向上面的元素添加 role="navigation"，有助于增加可访问性。

4.向 div 元素添加一个标题 class .navbar-header，内部包含了带有 class navbar-brand 的 a 元素。这会让文本看起来更大一号

5.为了向导航栏添加链接，只需要简单地添加带有 class .nav、.navbar-nav 的无序列表即可。
